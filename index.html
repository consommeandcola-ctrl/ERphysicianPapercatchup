PubMed（文献検索・Abstract取得）は通常 http://localhost 経由で開けば API キー不要で動きます。ブラウザの制限で fetch が失敗する場合は下記のローカルサーバーを使ってください。

解決策①（推奨）：Pythonローカルサーバーで起動
このHTMLファイルがあるフォルダで以下を実行：
python -m http.server 8080
その後ブラウザで http://localhost:8080/index.html を開く

解決策②：VSCode / Cursor の Live Server拡張を使う
ファイルを右クリック → "Open with Live Server"

解決策③：Chromeフラグ（簡易）
chrome.exe --allow-file-access-from-files で起動
